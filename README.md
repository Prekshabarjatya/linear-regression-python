# linear-regression-python

A structured learning repository for linear regression using Python and Google Colab notebooks.

## Project Overview

This repository provides a step-by-step learning path for linear regression, from synthetic data generation to model evaluation. The notebooks are designed for beginners who want hands-on practice with core tools in the Python data stack.

## Learning Objectives

By completing these notebooks, you will learn how to:

- Generate synthetic linear data with noise
- Visualize variable relationships using Seaborn
- Fit a simple linear regression model with Statsmodels OLS
- Interpret model coefficients and statistical outputs
- Evaluate regression performance using standard metrics

## Notebook Sequence

1. `/notebooks/01_data_generation.ipynb` - Create synthetic regression data.
2. `/notebooks/02_visualization_seaborn.ipynb` - Explore data visually with Seaborn.
3. `/notebooks/03_simple_linear_regression_statsmodels.ipynb` - Fit a simple OLS model.
4. `/notebooks/04_model_interpretation.ipynb` - Interpret slope, intercept, p-values, and R-squared.
5. `/notebooks/05_evaluation_metrics.ipynb` - Compute MAE, MSE, RMSE, and R².
